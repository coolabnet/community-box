  • ✅ Vite config sets base="/community-box/" in production

How it works:

  1. User navigates to https://username.github.io/community-box/questionnaire
  2. GitHub Pages returns 404.html (no physical file exists)
  3. 404.html script converts URL to https://username.github.io/community-box/?/questionnaire
  4. GitHub Pages serves index.html (query string ignored)
  5. index.html script converts back to /community-box/questionnaire
  6. React Router handles the route with basename="/community-box"
